# Bulkly V3: Omnichannel Marketing Hub

Bulkly is a production-ready, intelligent communication solution designed specifically for travel and migration agencies. It transforms manual outreach into a high-scale, AI-driven operation by integrating smart messaging, lead management, and multi-platform connectivity into a single premium interface.

## 🚀 Key Features

*   🤖 **AI Smart Replies**: Integrated with **Google Gemini** for intelligent, context-aware customer interactions.
*   🌐 **Omnichannel Ecosystem**: Unified support for **WhatsApp**, **Instagram**, **Facebook Messenger**, **Telegram**, and **Threads**.
*   🎨 **Premium UI**: Modern, dark-mode interface designed for high-performance marketing workflows using CustomTkinter.
*   📢 **Thematic Campaigns**: Dynamic template generation with 'Friendly', 'Urgent', and 'Professional' themes.
*   👥 **Community Builder**: Batch management tools for growing WhatsApp groups and lead communities.

## 🛠️ Technical Stack

- **Core**: Python 3.14
- **UI Framework**: CustomTkinter (Premium Dark Theme)
- **AI Engine**: Google Generative AI (Gemini Pro)
- **Automation**: Selenium WebDriver (for WhatsApp Web)
- **Distribution**: Compiled `.exe` via PyInstaller

## 📂 Project Structure

- `Bulkly.exe`: The finalized, standalone application.
- `whatsapp_bulk_sender/`: The core source code directory.
  - `core/`: Automation logic and platform connectors.
  - `ui/`: Modern interface components.
  - `data/`: Template and settings management.

## ⚙️ Setup & Usage

1. **Launch**: Open `Bulkly.exe`.
2. **Import**: Load your leads via Excel.
3. **Configure**: Set your Gemini API key and company details in the Settings panel.
4. **Automate**: Select your active platform and start the outreach campaign.
