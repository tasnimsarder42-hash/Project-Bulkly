# Sonali Sourav Migration & Visa - Bulk WhatsApp Sender

এটি সোনালী সৌরভ মাইগ্রেশন এ্যান্ড ভিসা কোম্পানির জন্য তৈরি একটি অফলাইন, অটোমেটেড বাল্ক হোয়াটসঅ্যাপ সেন্ডার অ্যাপ্লিকেশন। এই অ্যাপ্লিকেশনটি এক্সেল থেকে ডেটা নিয়ে স্বয়ংক্রিয়ভাবে ক্লায়েন্টদের জন্য কাস্টমাইজড মেসেজ তৈরি করে এবং ক্রোম ব্রাউজারের মাধ্যমে পাঠায়।

## ফিচারসমূহ
- **এক্সেল ইমপোর্ট:** সহজেই .xlsx ফাইল থেকে ফোন নম্বর ও ক্লায়েন্টের তথ্য ইমপোর্ট করা যায়।
- **স্বয়ংক্রিয় মেসেজ জেনারেটর:** প্রতিটি ক্লায়েন্টের জন্য আলাদা আলাদা মেসেজ তৈরি হয় যাতে হোয়াটসঅ্যাপ ব্যান না করে।
- **অ্যান্টি-ব্যান সিস্টেম:** মেসেজ পাঠানোর মাঝে অটোমেটিক বিরতি এবং দীর্ঘ সময় পর পর ব্রেক নেয়ার সুবিধা রয়েছে।
- **ড্যাশবোর্ড:** কতগুলো মেসেজ পাঠানো হয়েছে, কতগুলো ফেইল হয়েছে তার লাইভ স্ট্যাটাস দেখা যায়।
- **লগ সেভিং:** অ্যাপ হঠাৎ বন্ধ হয়ে গেলেও আগের অবস্থা থেকে আবার শুরু করা যায়।

## সেটআপ ও চালানোর নিয়ম (ডেভেলপারদের জন্য)

১. **পাইথন ইন্সটল:** আপনার কম্পিউটারে Python 3.10 বা তার বেশি ভার্সন ইন্সটল থাকতে হবে।
২. **রিকোয়ারমেন্টস ইন্সটল:** টার্মিনাল বা কমান্ড প্রম্পট ওপেন করে নিচের কমান্ড দিন:
   ```bash
   pip install -r requirements.txt
   ```
৩. **অ্যাপ রান করা:** 
   ```bash
   python main.py
   ```

## কীভাবে ব্যবহার করবেন?

১. অ্যাপটি চালু হলে বাম দিকের মেনু থেকে **Leads (লিডস)**-এ ক্লিক করুন।
২. **"Import Excel (এক্সেল ইমপোর্ট করুন)"** বাটনে ক্লিক করে আপনার লিড এক্সেল ফাইলটি সিলেক্ট করুন।
৩. লিডগুলো টেবিল-এ দেখা গেলে **Send (পাঠান)** মেনুতে যান।
৪. **"Start Sending (পাঠানো শুরু করুন)"** বাটনে ক্লিক করুন। 
৫. একটি ক্রোম ব্রাউজার ওপেন হবে এবং `web.whatsapp.com` এ যাবে। যদি আপনার হোয়াটসঅ্যাপ লগইন করা না থাকে, তাহলে প্রথমে **QR কোড স্ক্যান করে লগইন করুন**।
৬. লগইন সম্পূর্ণ হলে অ্যাপটি নিজে থেকেই মেসেজ পাঠানো শুরু করবে। আপনি চাইলে **Pause** বাটনে ক্লিক করে থামাতে পারেন।

## অ্যান্টি-ব্যান সতর্কতা এবং সেরা অভ্যাস (Best Practices)

বাল্ক মেসেজ পাঠানোর সময় হোয়াটসঅ্যাপ অ্যাকাউন্ট ব্যান হওয়া থেকে বাঁচাতে নিচের নিয়মগুলো মেনে চলুন:
- **নতুন অ্যাকাউন্ট ব্যবহার করবেন না:** অন্তত ৬ মাস পুরনো এবং আগে থেকে চ্যাট হিস্ট্রি আছে এমন নম্বর ব্যবহার করুন।
- **সেটিংস কনফিগার করুন:** Settings প্যানেলে গিয়ে Min Delay ১০ সেকেন্ড এবং Max Delay ৩০ সেকেন্ডের উপরে রাখুন।
- **ব্রেক নিন:** প্রতি ২৫-৩০ মেসেজ পাঠানোর পর ২-৫ মিনিটের লম্বা ব্রেক দিন।
- **অতিরিক্ত মেসেজ পাঠাবেন না:** একদিনে ২৫০-৩০০ এর বেশি মেসেজ পাঠানো থেকে বিরত থাকুন।
- **রিপ্লাই দিন:** যাদেরকে মেসেজ পাঠাচ্ছেন তারা যদি রিপ্লাই দেয়, তবে দ্রুত উত্তর দিন। এতে হোয়াটসঅ্যাপ অ্যালগরিদম অ্যাকাউন্টটিকে নিরাপদ মনে করবে।

## পোর্টেবল .exe তৈরি করা

যদি আপনি এই প্রজেক্টটিকে একটি সিঙ্গেল .exe ফাইলে রূপান্তর করতে চান যা যেকোনো উইন্ডোজ পিসিতে পাইথন ছাড়াই চলবে, তবে নিচের কমান্ডটি ব্যবহার করুন:
```bash
pip install pyinstaller
pyinstaller build.spec
```
তৈরিকৃত ফাইলটি `dist` ফোল্ডারের ভেতর পাওয়া যাবে।
